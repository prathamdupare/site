# Pratham Dupare - Personal Portfolio

A terminal-inspired personal portfolio website showcasing my projects, experience, and skills.

## Overview

This is my personal portfolio site built with vanilla HTML and CSS. It features a terminal/editor aesthetic with a sidebar navigation and main content area. The design is inspired by code editors like Vim and features syntax highlighting colors.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with CSS variables, responsive design
- **No JavaScript frameworks** - Pure vanilla JS for interactions

## Features

- Terminal/editor-inspired design
- Sections: Whoami, Projects, Experience, Skills, Contact
- Responsive layout with mobile hamburger menu
- Smooth scrolling navigation
- Active section highlighting
- Blink cursor animations

## Sections

- **whoami** - Profile information (name, location, editor, environment)
- **projects** - Featured open-source work
- **experience** - Work history (FoCDoT, Fosspage, Upwork)
- **skills** - Technical skills and tools
- **contact** - Social links (GitHub, LinkedIn, Twitter/X, Instagram)
